1.Picker

Drag the triangle for (W, B, C); drag the ring for hue. The C vertex tracks the selected hue.

2.Quantized picker

Only the ternary-grid points at resolution N are clickable.

3.2D grid

Rows = chroma band k, cols = lightness index i. Top-right = white, top-left = black, bottom-left = pure hue.

4.Arc-based scales

Five arcs run black → max-C midpoint → white at peak-chroma levels 0–100%. Each arc carries N+1 tokens spaced by arc length.
