Hue-name survey — clusters along the hue wheel

Named colors from 7 digital design systems + CSS named colors, projected onto HSL hue (sRGB-based, regular 60° spacing for primaries/secondaries — not OKLCH, which stretches red→yellow to ~81°). Each row is a name; glows show where that name appears, brighter where multiple sources agree. The bottom ribbon is a per-5°-bin vote of the most-used name among the filtered set — a first-draft canonical mapping. See design/color-system.md §14.
